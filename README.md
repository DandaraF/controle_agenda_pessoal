# Controle de agenda pessoal

## Objetivo
 - Desenvolver uma aplicação para controle de agenda pessoal.

## Tecnologias

- Vuejs3
- Axios

## Como executar

* Clone este repositório

* Instalar o npm:
  ```console
  npm install
  ```
* Starte o projeto:
  ```console
  npm run serve
  ```
